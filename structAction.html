Detailed Description

Represents an input action triggered by keyboard or mouse input, along with its current state.

Encapsulates an action with its current state (pressed, scroll, mouse motion, etc.). Used by scenes to process player input mapped to game actions.

Scene, InputKey, ScrollType, InputType
